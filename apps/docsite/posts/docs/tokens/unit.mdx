UICapsule unit system is based on one base value defined in the theme, which is `4px` in the default theme.
We generate multipliers from 1 to 10 to be used in the product directly.

```css
--rs-unit-x1
...
--rs-unit-x10

/* Usage */
.element {
  margin-top: var(--rs-unit-x4);
}
```

Sometimes these multiplier variables are not enough, so you can multiply the value yourself in CSS.

```css
.element {
  height: calc(var(--rs-unit-x1) * 40);
}
```
