title: Shadow
description: Shadow tokens help with communicating component elevation along the z-axis.
---

```preview
<ThemeProvider colorMode="light">
  <View backgroundColor="neutral-faded" padding={8} borderRadius="large" width="300px">
    <Card elevated>
      <View gap={2} direction="row" align="center">
        <Avatar initials="A" color="primary" />
        <View.Item grow>
          <Text variant="body-strong-2">UICapsule</Text>
          <Text color="neutral-faded">10,873 followers</Text>
        </View.Item>
      </View>
    </Card>
  </View>
</ThemeProvider>
```

UICapsule supports two shadow tokens that demonstrate the level of elevation of the component.

```css
--rs-shadow-base
--rs-shadow-elevated

/* Usage */
.element {
  box-shadow: var(--rs-shadow-base);
}
```

Base shadow is used to elevate the elements that initially don't have any elevation, for example, in `elevated` `Card`.
Elevated shadow is used for elements that are displayed on top of other elements, like in `DropdownMenu`.

Shadows are usually used with base and elevated background color tokens: `--rs-color-background-base` and `--rs-color-background-elevated`.
Since shadows are less prominent in dark mode, these color tokens will adjust the element's background color to communicate the elevation to the user.
