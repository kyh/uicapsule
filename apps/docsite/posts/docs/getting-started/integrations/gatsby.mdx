---

## Create your project

Start by creating your Gatsby project if you don't have one already.
Using [Gatsby init](https://www.gatsbyjs.com/docs/quick-start/) should speed up this process:

```bash
npm init gatsby
```

## Install UICapsule

Download the latest UICapsule release from the [Releases page](/content/changelog) and save the `.tgz` file into your project.
Then install the package using the file path.
For example, if you save the package as `./vendor/@uicapsule/components.tgz` in the project root folder, you can run:

```json
// npm
npm install ./vendor/@uicapsule/components.tgz

// yarn
yarn add ./vendor/@uicapsule/components.tgz
```

## Add PostCSS config

Install `gatsby-plugin-postcss` and add a PostCSS config from UICapsule through its options:

```js
import postcss from "@uicapsule/components/postcss";

const config = {
  ...
  plugins: [
    {
      resolve: "gatsby-plugin-postcss",
      options: {
        postCssPlugins: postcss.config.plugins,
      },
    },
  ],
};

export default config;
```

## Start the application

Run your build process with `npm run develop`.

```bash
npm run develop
```

## Start using UICapsule components

Import some of the core components from `@uicapsule/components/bundle` along with `@uicapsule/components/bundle.css`:

```code
import { Button, Container, UICapsule } from "@uicapsule/components/bundle";
import "@uicapsule/components/bundle.css";
import @uicapsule/componentsTheme from "@uicapsule/components/themes/@uicapsule/components/theme.css";

const App = () => {
  return (
    <UICapsule theme="@uicapsule/components">
      <Container width="652px">
        <Button href="/">Get started</Button>
      </Container>
    </UICapsule>
  );
};
```

You can find more details in our `gatsby-starter` example on Github.
If you want to learn more about components - search for the relevant components documentation in the sidebar.
